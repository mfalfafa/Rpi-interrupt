This is a project to detect goods in pin 25 in raspi 3 using interrupt.
You can use either Python script or Javascript to detect goods in Rpi 3 pin.

How to use :
1. Git clone in your Raspi 3
2. If you using Javascript, execute "npm install onoff" in your Raspi terminal and execute script by type "node rpi-interrupt.js"
3. If you using Python script, execute "pip3 install Rpi.GPIO" and execute script by type "python3 rpi-interrupt.py"
4. Enjoy it
